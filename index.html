Course Description

This course provides a comprehensive introduction to artificial intelligence designed specifically for medical and veterinary students with no computer science background. Students will learn fundamental AI concepts, gain hands-on experience with modern AI development tools, and apply these techniques to real-world healthcare applications and life sciences research.

Key Learning Objectives:

  • Develop a big-picture understanding of AI/ML techniques and their applications
  • Master modern AI development tools (Cursor, GitHub Copilot, ChatGPT, Claude)
  • Build practical applications: web apps and interactive tools for healthcare
  • Reproduce and understand AI research models in life sciences
  • Critically evaluate AI systems and their limitations in medical contexts

Prerequisites

No programming experience required. Students should have:

  • Basic understanding of medical/veterinary sciences
  • Comfort with using computers and learning new software
  • Willingness to experiment and learn through hands-on practice

Schedule & Syllabus

Week 1: Introduction - AI in Medicine

Topics: Course overview, what is AI?, AI in healthcare today, success stories and failures

Lab: Setting up development environment, introduction to Cursor AI editor

Reading: Esteva et al. "A guide to deep learning in healthcare" (Nature Medicine, 2019)

Week 2: AI Fundamentals I - Machine Learning Basics

Topics: Supervised vs. unsupervised learning, classification vs. regression, training/validation/testing, overfitting

Lab: Using AI tools to analyze medical datasets (diagnosis prediction)

Reading: Selected chapters from "Machine Learning Yearning" by Andrew Ng

Week 3: AI Fundamentals II - Neural Networks & Deep Learning

Topics: Neural networks intuition, CNNs for medical imaging, how models learn

Lab: GitHub Copilot hands-on, building a simple image classifier

Reading: Goodfellow et al. Deep Learning (selected chapters)

Week 4: AI Development Tools Ecosystem

Topics: LLM-powered coding assistants, ChatGPT/Claude for research, Hugging Face and open-source models

Lab: Exploring pre-trained models, prompt engineering for medical tasks

Due: Assignment 1 - AI Tools Mastery Portfolio

Week 5: Web Development Basics with AI

Topics: HTML/CSS/JavaScript fundamentals, using AI to accelerate development, responsive design

Lab: Building your first medical information website with AI assistance

Reading: Web development resources (provided)

Week 6: Interactive Web Applications

Topics: JavaScript frameworks (React basics), APIs and data integration, deploying web apps

Lab: Creating an interactive symptom checker or medical calculator

Reading: Modern web development tutorials

Week 7: Computer Vision in Medical Imaging

Topics: CNNs deep dive, medical image analysis (X-rays, CT, MRI, pathology), segmentation and detection

Lab: Using pre-trained models for medical image analysis

Reading: Litjens et al. "A survey on deep learning in medical image analysis"

Due: Assignment 2 - Healthcare Web Application

Week 8: Natural Language Processing in Healthcare

Topics: Text analysis, clinical notes processing, medical literature mining, chatbots for patient interaction

Lab: Building a medical literature summarizer with LLMs

Reading: Recent papers on clinical NLP

Week 9: AI in Life Sciences Research I

Topics: Drug discovery, protein structure prediction (AlphaFold), genomics and bioinformatics

Lab: Reproducing a published model - getting started, data preparation

Reading: Jumper et al. "Highly accurate protein structure prediction with AlphaFold"

Week 10: AI in Life Sciences Research II

Topics: Single-cell analysis, medical imaging research, reproducibility in AI research

Lab: Model reproduction workshop - implementation and validation

Reading: Selected research papers on AI in life sciences

Week 11: AI for Veterinary Medicine

Topics: AI applications in animal health, comparative medicine, challenges unique to veterinary AI

Lab: Adapting human medicine models for veterinary applications

Reading: Recent papers on AI in veterinary medicine

Week 12: Ethics, Bias, and Interpretability

Topics: Algorithmic bias in healthcare, fairness and equity, explainable AI, FDA regulations and medical AI

Lab: Analyzing bias in medical AI systems, interpretability tools

Reading: Obermeyer et al. "Dissecting racial bias in an algorithm used to manage the health of populations"

Due: Assignment 3 - Research Model Reproduction

Week 13: Advanced Topics & Current Research

Topics: Federated learning for healthcare, multimodal AI, AI for clinical trials, future directions

Lab: Final project work session

Reading: Recent papers from top medical AI conferences

Week 14: Final Project Presentations

Topics: Student project presentations and course wrap-up

Due: Final Project

Major Assignments

Assignment 1: AI Tools Mastery Portfolio (15%)

Due: Week 4

Objective: Demonstrate proficiency with modern AI development tools and understand their strengths and limitations.

Deliverables:

  • Tool Exploration Report: Document your experience with Cursor, GitHub Copilot, ChatGPT, and Claude. Include specific examples of tasks where each tool excelled or struggled.
  • Mini-Projects: Complete 3 small coding tasks using AI assistance:
    • Data analysis script for a medical dataset
    • Simple diagnostic decision tree implementation
    • Medical terminology quiz game
  • Reflection: 2-page essay on how AI tools change the software development process and their potential impact on medical informatics

Skills Developed: AI-assisted coding, prompt engineering, critical evaluation of AI outputs, debugging AI-generated code

Assignment 2: Healthcare Web Application (25%)

Due: Week 7

Objective: Build a functional, user-friendly web application that serves a healthcare-related purpose.

Requirements:

  • Create a complete web application with frontend and basic backend logic
  • Must incorporate at least one AI/ML component (e.g., prediction model, NLP feature, image analysis)
  • Professional UI/UX design appropriate for medical context
  • Deployed and accessible via web URL

Example Projects:

  • Interactive medical calculator (e.g., drug dosing, risk scores)
  • Symptom checker with AI-powered triage suggestions
  • Medical image viewer with annotation and AI analysis tools
  • Patient education platform with personalized content
  • Veterinary clinic management dashboard

Deliverables:

  • Live web application (URL)
  • Source code repository (GitHub)
  • 5-page documentation including design decisions, AI tool usage, and user guide
  • 5-minute video demo

Assignment 3: Research Model Reproduction (25%)

Due: Week 12

Objective: Reproduce a published AI research model in life sciences, understanding the full pipeline from data to results.

Process:

  1. Paper Selection: Choose from provided list or propose a paper (must be approved by instructor)
    • Focus areas: medical imaging, drug discovery, genomics, clinical prediction
    • Paper should include accessible dataset and clear methodology
  2. Reproduction: Implement the model and reproduce key results
    • You may use AI tools extensively for this assignment
    • Document all code and decisions
    • Compare your results with published results
  3. Extension: Add one novel element:
    • Test on a different dataset
    • Try different hyperparameters or architecture modifications
    • Create visualization tools for the results
    • Analyze model interpretability

Deliverables:

  • Complete code repository with clear documentation
  • 8-10 page report in conference paper format including:
    • Original paper summary
    • Reproduction methodology and challenges
    • Results comparison
    • Novel extension and findings
    • Critical analysis of reproducibility
  • Jupyter notebook walkthrough

Suggested Papers: Will be provided with detailed guidelines and datasets

Final Project: Choose Your Own Adventure (15%)

Due: Week 14

Objective: Apply course concepts to a project of personal interest relevant to your field.

Options:

  1. Application Development: Build a more ambitious web/mobile app or game with AI features
  2. Research Project: Conduct a small research study using AI on a dataset of your choice
  3. Educational Tool: Create an interactive learning tool for medical/vet students
  4. Literature Review + Demo: Comprehensive survey of AI in a specific medical domain with prototype implementation

Deliverables:

  • Final product (application, research paper, or tool)
  • 10-minute presentation to class
  • Project website/documentation
  • Reflection on learning journey and future applications
